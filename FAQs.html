How many models are in the repository?

The repository currently contains 92 models, divided into: 27 aorta models, 22 aortofemoral models, 4 cerebrovascular models, 12 congenital heart diseased models, 11 coronary models, and 16 pulmonary models.

What type of image data is contained in the repository?

Each project contains medical image data in .vti format.

How do I use this website?

The project files can be accessed from the Repository page. Each model can be downloaded by clicking on the model name appearing when hovering on the corresponding picture. The projects are compatible with SimVascular; please refer to the Documentation for more information on how to open and use them.
