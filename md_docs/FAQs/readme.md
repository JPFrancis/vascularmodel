## What type of image data is contained in the repository?

Each project contains medical image data in `.vti` format.

## How do I use this website?

The project files can be accessed from the <a href="dataset.html" class="link" target="_blank">Repository</a>. More information on how to navigate the dataset can be found in the <a href="tutorial.html" class="link" target="_blank">Tutorial</a> page. The projects are compatible with SimVascular; please refer to the
<a href="documentation.html" class="link" target="_blank">Documentation</a>
for more information on how to open and use them.
